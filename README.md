## ✦ ¿Qué es Ricker72 Web Hub?

**Ricker72 Web Hub** es la **plantilla oficial** donde se anunciarán todas las actualizaciones, builds, notas de versión y detalles de desarrollo de mis proyectos.  
Actúa como un centro de control visual que integra en tiempo real la información de mis repositorios de GitHub, ofreciendo una experiencia futurista e interactiva.

> 🚧 **Este sitio evoluciona constantemente**. Aquí verás reflejados los últimos cambios de [Npc-Maker](https://github.com/ricker72/Npc-Maker), [agente_rme](https://github.com/ricker72/agente_rme) y [Naxed-Server-Revamp-15-](https://github.com/ricker72/Naxed-Server-Revamp-15-), además de adelantos exclusivos.

---

## 🎯 Funcionalidades principales

- **Conexión en vivo con GitHub** – Los datos de cada proyecto (estrellas, forks, descripción, lenguaje y README) se obtienen dinámicamente mediante la API oficial.
- **Efectos visuales de alto impacto** – Fondo con partículas en forma de estrellas doradas, estrellas fugaces, cursor con estela luminosa y animaciones de desplazamiento (AOS).
- **Tarjetas 3D interactivas** – Cada proyecto tiene una tarjeta con efecto *tilt* al pasar el ratón, borde neón y botón para desplegar el README completo sin salir de la página.
- **Terminal de perfil (opcional)** – Integraba un acceso rápido al README de mi perfil de GitHub en tiempo real.
- **Diseño responsive** – Adaptable a móviles, tablets y escritorio, manteniendo la estética negro + dorado.
- **Carga asíncrona** – Spinner dorado mientras se recuperan los datos de GitHub, ofreciendo una experiencia fluida.
